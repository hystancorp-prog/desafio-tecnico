# Desafio Técnico

Repositório com os três desafios técnicos desenvolvidos em Java e React.

## Desafios

### Desafio 1 — Sistema de Venda e Fidelidade de Posto de Combustível
Algoritmo em Java para gerenciar venda de combustíveis com regras de fidelidade e desconto.

### Desafio 2 — Sistema de Divisão de Gorjetas
Algoritmo em Java para dividir gorjetas entre garçons com truncamento de centavos.

### Desafio 3 — GitHub Explorer
Aplicação web em React que consome a API do GitHub para buscar perfis e repositórios.
Veja o README completo em `/desafio-3/README.md`.

## Tecnologias
- Java
- React + Vite
- Tailwind CSS
- React Router DOM
- GitHub REST API
