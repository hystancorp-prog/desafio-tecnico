# GitHub Explorer

Aplicação web para buscar perfis e repositórios do GitHub.

## Funcionalidades

- Busca de usuários do GitHub
- Exibição de perfil com avatar, bio, seguidores e seguidos
- Listagem de repositórios públicos ordenados por estrelas
- Ordenação por estrelas, nome ou data de atualização
- Página de detalhes do repositório
- Persistência da última busca ao navegar entre páginas

## Tecnologias

- React
- Vite
- Tailwind CSS
- React Router DOM
- GitHub REST API

## Como rodar

**Pré-requisitos:** Node.js instalado

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev
```

Acesse `http://localhost:5173` no navegador.
